## 📍 내 위치 기반 공공 와이파이 정보 제공 웹서비스

서울시 오픈 API 활용하여 JAVA 기반 다이나믹 웹 구현 

<br>

## 📝 기능 구현

- 서울시 공공 와이파이 정보 가져오기
- 내 위치와 가장 가까운 위치에 있는 와이파이 정보 20개 보여주기
- 조회했던 위치 정보의 히스토리 보여주기
